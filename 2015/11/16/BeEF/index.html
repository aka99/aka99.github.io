BeEF

发表于 2015-11-16

BeEF是浏览器攻击框架的简称，是一款专注于浏览器端的渗透测试工具。

BeEF攻击浏览器依赖Javascript

主流浏览器均支持JS。

能控制浏览器完成各种任务：

  • 浏览页面截屏
  • 利用浏览器主机作为代理渗透内网
  • 社交工程攻击

BeEF使用基础

如何hook 浏览器？

  • 中间人攻击 MITMf
  • 钓鱼、垃圾邮件
  • 恶意网站注入JS

如何unhook 感染的浏览器？

  • run unhook(Commands Browser Unhook)

如何持久化感染的浏览器？

  • 集成Metaspolit
  • hook 浏览器当前domain
  • 运行 Metasploit 模块进行攻击测试
