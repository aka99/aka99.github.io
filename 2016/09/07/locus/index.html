HTTPS 网站压力测试

发表于 2016-09-07 | 分类于 技术工具

对于网站启用了HTTPS，担心性能下降，需要了解一下处理能力，故要压力测试一下，一开始想到了apache ab测试工具，但是对于复杂的场景ab无能无力，基于python的开源工具locust比较不错，测试登陆、rest api都可以。

特点：

  • 测试用例自编python脚本
  • 支持分布式部署测试，一台作为master主控，其他多台salve。

实例：

  • django 网站登陆及页面访问压力测试
  • django rest api 压力测试，为了并发测试，提前获取到jwt token，填入脚本，直接跳过登陆
