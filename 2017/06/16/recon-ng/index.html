recon-ng 使用更新

发表于 2017-06-16 | 分类于 OSINT

OSINT 工具 recon-ng

安装

  • Clone the Recon-ng repository.
    • git clone https://LaNMaSteR53@bitbucket.org/LaNMaSteR53/recon-ng.git
  • Change into the Recon-ng directory.
    • cd recon-ng
  • Install dependencies.
    • pip install -r REQUIREMENTS
  • Launch Recon-ng.
    • ./recon-ng
  • Use the “-h” switch for information on runtime options.
    • ./recon-ng -h

一般使用流程

设置api: keys add

创建工作区：workspaces add test
添加域名：add domains tesst.cn

查找模块：search name
使用模块：use name
查看模块信息：show info
查看需要设置的参数：show options
设置参数：set SOURCE …
