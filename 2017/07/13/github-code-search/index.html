gitub code search

发表于 2017-07-13 | 分类于 数据安全

今日，一则新闻很奇葩，锤子员工在GitHub上提交的文件被发现竟然包含坚果Pro的激活数量。新闻链接:锤子员工在GitHub泄密坚果Pro真实销量

员工在github 上传敏感信息对组织的数据代码防泄漏工作来说是个噩梦，如何及时有效进行监控，是组织必备的安全措施之一。

目前国内有好几家公司提供github敏感信息监控服务，其中携程安全部免费提供了github scan云监控服务，只要是甲方企业通过申请后，就可以使用，使用有限制，只能监控10个关键字。这里介绍一款开源监控工具GitMiner，GitMiner是一款轻量级的github监控工具，Github 地址

使用准备

需要用户登陆github网站的token，方法是使用chrome 查看user_session，我是保存在 git_miner.py 中，也可以每次运行时指定。

示例:查看etc shadow

执行
python git_miner.py --query 'filename:shadow path:etc' -m root

结果看请本文底部录像

示例: wordpress

执行
python git_miner.py -q 'filename:wp-config extension:php FTP_HOST in:file ' -m wordpress -o result.txt

示例: joomla

执行
python git_miner.py --query 'filename:configuration extension:php "public password" in:file' -m joomla

示例: 自定义

编辑 config\parse.json 添加一个模块

执行
python git_miner.py -q 'cpic.com password language:java' -m my
